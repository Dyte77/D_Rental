
# Rental Connect

A cross-platform mobile application connecting landlords directly with tenants
searching for rental housing — built to eliminate broker scams and overpricing
by giving tenants direct, verified access to real listings.

## Structure

- `/backend` — Node.js + Express REST API, PostgreSQL database
- `/mobile` — Flutter cross-platform mobile app (Android + iOS)
- `/docs` — system design documentation (architecture, ERD, system logic)

## Status

🚧 In active development.

## Tech stack

- **Mobile:** Flutter (Dart)
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **Auth:** JWT + OTP phone verification
- **Storage:** Cloud object storage for listing images
